# CS180SteamDatabaseWeb

## Overview
We created a web based data analytics app using the steam dataset from https://www.kaggle.com/datasets/nikdavis/steam-store-games. Using the dataset, we are able to search, delete, add, and update entries. We performed analytics on the dataset, such as calcuating game's average ratings and displaying the developer breakdowns.
